Site Name: PC Power

Site Purpose

This site is a one-stop shop to help first time PC builders by providing parts recommendations and a guide on how to build a PC.

They will be able to pick from different parts using buttons and forms to get personalized recommendations for them.

Scenarios

Question: What is the best PC I can build for under $1000?

Question: What parts would work best if I mainly use my computer for video editing?

Question: How do I know how much system memory I need?

Answer to all of these: These will all be provided on the parts page. There will also be a page showing how to build a PC step by step.
